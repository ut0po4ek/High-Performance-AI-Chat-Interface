# High-Performance AI Chat Interface (SPA)

SPA-чат интерфейс для LLM-платформы на React + TypeScript + Zustand + Tailwind + Vite.
Поддерживает большой объём истории и быстрый streaming без фризов, виртуализированный список и Markdown.

## Требования
- Node.js 18+ (рекомендуется 20+)
- npm (или pnpm/yarn — если используешь, команды будут аналогичны)

## Установка
```bash
npm install
```

### Запуск в dev mode
```bash
npm run dev
```
